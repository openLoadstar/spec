# 🌐 LOADSTAR Addressing Convention

- **M** (Map): `M://path/id`
- **W** (WayPoint): `W://path/id`

* 모든 경로는 루트(/)로부터의 절대 경로를 지향한다.

## 물리 파일 경로 변환 규칙

논리 주소는 다음 규칙에 따라 파일 시스템 경로로 변환된다.

```
TYPE://seg1/seg2/.../id
→ .loadstar/[TYPE_DIR]/seg1.seg2...id.md
```

| 논리 주소 | 물리 파일 경로 |
| :--- | :--- |
| `W://root/cli/cmd_show` | `.loadstar/WAYPOINT/root.cli.cmd_show.md` |
| `M://root/cli` | `.loadstar/MAP/root.cli.md` |

## TYPE_DIR 매핑

| 타입 접두사 | 폴더명 |
| :--- | :--- |
| `M` | `MAP` |
| `W` | `WAYPOINT` |
